
Website ini dirancang untuk memenuhi kebutuhan komunitas club motor, memberikan informasi yang komprehensif bagi para anggotanya serta para pengunjung yang tertarik. Struktur website ini mencakup empat halaman utama, yaitu halaman utama, halaman artikel, halaman gallery, dan halaman event. Setiap halaman dirancang agar user-friendly dengan memanfaatkan teknologi HTML, Bootstrap, dan JavaScript dasar untuk tampilan yang responsif dan dinamis.

1. Halaman Utama (Homepage)
Halaman utama berfungsi sebagai pintu masuk ke seluruh konten yang tersedia di website. Di sini, pengunjung dapat menemukan informasi dasar tentang club motor, visi dan misi, sejarah pembentukan club, serta event atau kegiatan terbaru yang diadakan oleh club. Dengan menggunakan Bootstrap, halaman utama tampil menarik dan responsif, menyesuaikan dengan berbagai ukuran layar perangkat, baik desktop maupun mobile. Efek-efek dasar JavaScript diterapkan untuk memperkaya pengalaman pengguna, seperti efek hover dan scroll pada elemen-elemen penting.

2. Halaman Artikel
Halaman artikel menyediakan berbagai konten terkait dunia otomotif dan club motor. Artikel-artikel ini dapat mencakup topik seperti tips berkendara, ulasan motor terbaru, perjalanan touring yang dilakukan oleh anggota club, hingga informasi teknis mengenai motor. Dengan memanfaatkan grid layout dari Bootstrap, setiap artikel disusun dalam bentuk card yang menarik dan memudahkan navigasi bagi pengguna. Pengguna dapat memilih artikel tertentu untuk dibaca lebih lanjut, yang ketika diklik akan membuka artikel secara penuh di halaman baru.

3. Halaman Gallery
Pada halaman gallery, pengunjung dapat melihat dokumentasi visual kegiatan-kegiatan club motor. Gallery ini menyajikan foto-foto dan video dari kegiatan touring, pertemuan club, hingga event khusus yang diselenggarakan oleh komunitas. Bootstrap digunakan untuk membuat gallery dengan tampilan grid yang rapi dan responsif. Efek JavaScript sederhana diterapkan untuk menampilkan gambar dalam mode lightbox atau fullscreen ketika diklik, sehingga pengunjung dapat melihat foto-foto dengan lebih jelas.

4. Halaman Event
Halaman event berisi informasi lengkap mengenai event-event yang akan dan telah diselenggarakan oleh club. Setiap event dilengkapi dengan informasi tanggal, waktu, lokasi, dan deskripsi singkat. Penggunaan Bootstrap membuat tampilan jadwal event terstruktur dengan baik, sehingga memudahkan anggota maupun pengunjung umum untuk menemukan detail event yang mereka minati. JavaScript dasar digunakan untuk membuat countdown timer menuju tanggal event mendatang, menambah elemen interaktif yang menarik.

Secara keseluruhan, website club motor ini mengutamakan kemudahan navigasi dan pengalaman pengguna dengan memanfaatkan elemen-elemen HTML, styling dari Bootstrap, serta efek dinamis dari JavaScript dasar. Dengan desain yang responsif, website ini diharapkan dapat menjadi platform informasi yang informatif, menarik, dan mudah diakses bagi para penggemar otomotif.
